AlgaeID Monitoring Dashboard

AlgaeID Monitoring memantau kualitas udara di sekitar lampu alga dan efektivitasnya. Website ini juga berfungsi sebagai laporan keberlanjutan, memberikan informasi dampak lingkungan dari penggunaan lampu alga.

Pengguna dapat memantau data real-time dan mendapatkan wawasan mengenai kontribusi alga terhadap kualitas udara dan keberlanjutan lingkungan.

Untuk menambahkan item baru, tekan tombol plus (+) di bagian bawah layar.

AlgaeID Monitoring menggunakan sensor untuk mengukur kualitas udara di sekitar lampu alga. Data dikirim ke server dan ditampilkan di dashboard dalam bentuk grafik dan laporan.

Pengguna dapat melihat data real-time, mendapatkan notifikasi perubahan signifikan, dan mengunduh laporan untuk analisis.
